# **9.** ***\*单品采购单管理\****

## **9.1.** ***\*生成单品采购单\****

采购经理在分店订购单中，点击”生成单品采购单“。注意：单品采购单生成后不允许删除。

![img](assets/wps5AWFCO.jpg) 

 

点击”确定“

![img](assets/wpstZiqIQ.jpg) 

 

## **9.2.** ***\*单品采购单搜索\****

在输入框中输入搜索内容，并点击”搜索“按钮。

![img](assets/wps25yvRq.jpg) 

 

## **9.3.** ***\*单品采购单修改\****

未完成的单品采购单可以点击“编辑”，对单品采购单的备注进行修改，其余信息皆不允许修改。当单品采购单完成以后，所有信息都不允许修改，只允许被查看。

![img](assets/wpsu8bX8q.jpg) 

 

## **9.4.** ***\*单品采购单打印\****

移动端不支持打印功能。

在要打印的采购单处点击”编辑“或”“查看”。

![img](assets/wpskjLWaz.jpg) 

 

然后点击“打印”。

![img](assets/wpse0D3XC.jpg) 

 

## **9.5.** ***\*单品采购单分配\****

采购经理在单品采购单页面中，点击”采购单任务分配“。

![img](assets/wpszN8QfO.jpg) 

 

选择方案后，点击确定。

![img](assets/wpsslbKIM.jpg) 

 

## **9.6.** ***\*采购信息录入信息和采购完成\****

当采购单分配给采购员以后，采购员采购完货品需要对采购信息进行录入。采购员录入信息的操作如下：点击采购单对应的“编辑”按钮，在弹出的页面中录入以下信息。

![img](assets/wpsX6Cebf.jpg) 

 

录入后点击“确定”，保存录入信息。当所有信息录入完毕，采购员点击采购单对应的”完成“按钮，该采购单就完成了，其状态改为“采购完成”。 “采购完成”的单品采购单会被放入历史采购单中，且不允许再被修改。

## **9.7.** ***\*历史单品采购单查询\****

按如下操作查询历史单品采购单。

![img](assets/wpsIzmj0B.jpg) 

 

查询历史单品采购单时，可以附加日期条件。

 

![img](assets/wpsXhKLcx.jpg)
